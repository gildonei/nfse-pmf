# NFSe-PMF (Prefeitura Municipal de Florianópolis)
API não oficial para emissão de NFS-e para a PMF (Florianópolis)
Documentação oficial: [https://nfps-e-hml.pmf.sc.gov.br/home/](https://nfps-e-hml.pmf.sc.gov.br/home/).

### Dependências

* PHP >= 5.6

### Instalando o SDK

Se já possui um arquivo `composer.json`, basta adicionar a seguinte dependência ao seu projeto:

```json
"require": {
    "gildonei/nfse-pmf": "dev-master"
}
```

Com a dependência adicionada ao `composer.json`, basta executar:

```
composer install
```

Alternativamente, você pode executar diretamente em seu terminal:

```
composer require gildonei/nfse-pmf:dev-master
```
